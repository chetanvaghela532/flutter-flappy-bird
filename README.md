# 🐦 Flutter Flappy Bird

A modern Flappy Bird game built entirely with Flutter.

This project demonstrates how to build a smooth and responsive 2D mobile game using Flutter while following clean architecture and best development practices.

## ✨ Features

- 🐦 Smooth Bird Physics
- 🎯 Collision Detection
- 🎮 Endless Gameplay
- 🏆 High Score System
- ✨ Beautiful Animations
- 🔊 Sound Effects
- 📱 Android Support
- 🍎 iOS Support
- 🌐 Flutter Web
- 💻 Windows, macOS & Linux
- ⚡ Optimized Performance
- 🏗️ Clean Architecture
- 🎨 Responsive UI

## 🛠 Tech Stack

- Flutter
- Dart
- Custom Game Physics
- Canvas Rendering
- AnimationController
- CustomPainter

## 📂 Project Structure

```
lib/
├── core/
├── game/
├── models/
├── services/
├── widgets/
├── screens/
└── main.dart
```

## 🚀 Getting Started

```bash
git clone https://github.com/yourusername/flutter-flappy-bird.git

cd flutter-flappy-bird

flutter pub get

flutter run
```
